## The Odin Project Recipes
### repos/odin-recipes

This project is a simple website built as part of The Odin Project, focused on practicing the fundamentals of HTML and CSS. The goal was to create a clean and organized collection of favorite recipes while reinforcing core front-end concepts.

## About the Project

The website consists of a main index page that links to individual recipe pages. Each recipe includes basic structure elements such as headings, images, ingredients, and step-by-step instructions.

The project emphasizes:

<ul>
  <li>Semantic HTML structure</li>
  <li>Linking between pages</li>
  <li>Basic layout and styling with CSS</li>
</ul>

## Styling & Improvements

While following the Odin Project curriculum, I also explored some concepts beyond the platform to improve the visual presentation (<em>I know, I’m extremely anxious</em>).

This includes:
<ul>
  <li>Using Flexbox to better center and align content</li>
  <li>Applying gradients to create a more modern and visually appealing background</li>
  <li>Improving spacing, typography, and layout consistency</li>
  <li>Using a <a href="https://meyerweb.com/eric/tools/css/reset/" target="_blank">CSS reset</a> to ensure consistent styling and better layout control.</li>
</ul>

These additions helped transform a very basic layout into something more polished and enjoyable to navigate.
